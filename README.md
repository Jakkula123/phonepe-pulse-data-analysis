# PhonePe Pulse Data Analysis

## 📌 Overview

## Problem Statement
This project analyzes PhonePe transaction data to understand how digital payments are growing across India. The focus is on identifying high-performing states, user adoption trends, and transaction behavior that can help in making business decisions such as market expansion and improving user engagement.

---

## Dataset
- Source: PhonePe Pulse  
- Includes: transaction amount, transaction count, registered users, and payment types across states and years  

---

## Tools Used
- Python (data cleaning and preprocessing)  
- SQL (data analysis and aggregation)  
- Power BI (dashboard and visualization)  

---

## Approach
- Cleaned and structured raw data using Python  
- Performed aggregations and analysis using SQL  
- Built an interactive dashboard in Power BI to visualize trends and patterns  

---

## Key Insights

- Telangana (~₹83.3T), Karnataka (~₹81.3T), and Maharashtra (~₹80.7T) are the top contributors, showing strong digital payment adoption in these regions.

- Total transaction value grew from ~₹3.24T in 2018 to ~₹259.25T in 2024, indicating rapid expansion of digital payments across the country.

- Registered users increased from ~292 million in 2018 to ~2.23 billion in 2024, showing strong user adoption and trust in digital platforms.

- The average transaction value is ₹1468.53, suggesting that users are making frequent mid-value transactions rather than large payments.

- Merchant payments account for 55.35% of transactions, while peer-to-peer contributes 36.14%, indicating that business transactions dominate the ecosystem.
